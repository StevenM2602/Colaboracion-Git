# Sistema de Gestión de Facturación (Reto 3.1)

Este es un proyecto educativo que implementa un **CRUD completo** utilizando el patrón de diseño **MVC (Modelo-Vista-Controlador)** en PHP.

## Funcionalidades
- Gestión de **Clientes**: Alta, baja, modificación y listado.
- Gestión de **Artículos**: Control de catálogo.
- Gestión de **Facturas**: Creación de cabeceras y líneas de detalle.
- **Seguridad**: Encriptación de contraseñas y manejo de sesiones.
- **Exportación**: Generación de archivos PDF y Excel.

## Requisitos e Instalación
1. Clonar el repositorio en `xampp/htdocs`.
2. Importar la base de datos SQL proporcionada.
3. Ajustar las credenciales en `config.php`.
